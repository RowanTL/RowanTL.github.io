These are notes taken on Wednesday January 2026 in Dr. Golik's Probability and
Statistics course.

We talked about section 1.3 in the textbook, covering some set A given
some other set B.

$$
  P(A | B) = \frac{P(A \cap B)}{P(B)}
$$

Is basically it.
